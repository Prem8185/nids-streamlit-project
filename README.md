# Network Intrusion Detection System (NIDS)

A Machine Learning-based Network Intrusion Detection System using the NSL-KDD dataset.

## Features

* Multi-class attack detection (Normal, DoS, Probe, R2L, U2R)
* Logistic Regression, KNN, Decision Tree, and Random Forest comparison
* Streamlit web application
* CSV file upload and prediction
* Attack count visualization
